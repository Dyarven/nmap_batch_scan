Bash script made to quickly check for open ports on a list of IPs provided via text file with multithreading support.

## Requirements
- NMAP and a  dream


## Usage
### Run the Script
```bash
# Run as sudo (some commands require root to make changes)
bash nmap_batch_scanner.sh
```
